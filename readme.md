# directus

- B1: ở thư mục gốc Run "pnpm install"
- B2: Run "pnpm build"
- chạy 2 cái: 1 app folder, 2 api folder
- api folder:
- B3a: Run "cd api"
- B4a: Run "pnpm install"
- B5a: Run "pnpm run dev"
- app folder:
- B3b: Run "cd app"
- B4b: Run "pnpm install"
- B5b: Run "pnpm run dev --host"
